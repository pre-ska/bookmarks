    • Top toolbar

  • Top toolbar contains buttons for main roots, search box and drop-down menu

    • Bottom toolbar


  • if you have V7 Sessions you can save V7 session directly from V7 Bookmarks in the same way instead of bookmarks folder, just enable that feature in settings (on settings page of this extension)

  • Stats will open popup window with basic statistic, number of bookmarks and folders by roots
  • Empty Trash button is visible only when Trash panel is active, and only if there is something in Trash bin
  • restore deleted bookmarks from Trash with right click on them (single or multi-selection)
  • restored bookmars will go to All Bookmarks (same behaviour as Opera bookmarks manager)
  • zoom commands are on the bottom and very simple
  • use them for web pages, not the extension panel
  • +/- will zoom up/down while click on zoom indicator will reset zoom to 100%
  • because of the chromium architecture, this commands won't work on internal pages and Opera addons store
  • red/blue switch is for changing zoom scope
  • state of the switch is saved, and all zooming with this extension will act according to that state

  • red state is per-origin
  • if the switch is red, zoom behaves in classic way, all tabs from same origin/domain are affected and zoom is saved for next visit

  • blue state is per-tab
  • if the switch is blue, changing zoom (with extension) won't affect other tabs from same origin and zoom will NOT be saved for next visit, any navigation in that tab will cancel zoom made by per-tab scope... so it's only this-tab-this-time ...4t™ :P

  • access features from right click context menu (different for multiple selection, bookmark, folder, panel,search results...)
  • you can also sort bookmarks in folder (direct children only), from folder right click menu
  • hold CTRL and RIGHT click on selection to access alternative context menu
  • Export collection will create HTML page from selected bookmarks that you can view in any browser. Useful if you need to send large collection of links to someone...
  • Export to HTML will create HTML file from selected bookmarks that you can import in any browser (not meant to be rendered in tab). Useful if you need to backup or transfer to another browser just some bookmarks and folders.
  • open bookmark in current tab with single click or double click (set in settings)
  • middle click on bookmark will open it in new tab
  • middle click on folder will open all bookmarks from that folder (not from subfolders)
  • open bookmarks and folders in new tab(s) by dragging them out of panel.
